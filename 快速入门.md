# 🚀 快速入门指南

## 最简单的使用方式（3步）

### 第一步：创建虚拟环境
双击运行：**`setup_venv.bat`**

> 这会自动创建虚拟环境并安装所有依赖包

### 第二步：运行分析
双击运行：**`run_analysis_venv.bat`**

> 程序会自动分析数据并生成11张可视化图表

### 第三步：查看结果
打开目录：**`analysis_results/`**

> 里面包含所有生成的图表和分析报告

## 🆘 遇到问题？

### 问题1：双击脚本后闪退
**解决**：右键脚本 → 编辑 → 查看错误信息

### 问题2：提示找不到Python
**解决**：安装Python或将Python添加到系统PATH

### 问题3：依赖安装失败
**解决**：检查网络连接，或手动运行：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 💡 进阶使用

### 手动激活虚拟环境
双击：**`activate_venv.bat`**

然后在命令行中运行：
```bash
python film_review_analysis.py
```

### 自定义分析
编辑 `film_review_analysis.py` 文件，调整：
- 输出目录
- 图表样式
- 停用词表
- 分析模块

详细说明请查看 **README.md**
